Functional requirements

On each page load, randomly pick one icon file from the 12 listed and render it left of the wordmark in the header.

The icon must render with a solid fill #4B0082. If the source paths contain their own fills, override to #4B0082.

The wordmark in the header must render with a linearGradient fill (0% #4B0082, 50% #32DCFA, 100% #00FFAF).

The wordmark in the footer must render in solid white (#FFFFFF).

Use inline SVG for gradient control in the header wordmark, or pre-bake the gradient inside astacar_logo_font.svg. Avoid raster images.

Header container: icon and wordmark aligned center vertically, with 8–12px gap.

Sizes: icon height equals header wordmark cap-height; suggested classes: h-8 md:h-10 for both; width auto.

Footer: center or left align per site style; suggested classes: h-8 md:h-10, fill-white.

Accessibility

Both header and footer logos must have accessible text: wrap in a link with aria-label="Astacar — home".

Provide <title>Astacar</title> inside the SVGs or aria-hidden="true" if redundant with link label.
